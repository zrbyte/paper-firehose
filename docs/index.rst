Paper Firehose Documentation
============================

Paper Firehose filters, ranks, and summarizes research-paper RSS feeds. The project
ships both a Click-based CLI and Python APIs for running the individual pipeline
steps. These docs cover configuration concepts, operational workflows, and the
module-level API reference.
